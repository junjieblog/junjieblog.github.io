MacOS使用ClashX教程

2024-09-19

ClashX是MacOS系统即苹果电脑系统下的代理软件客户端，是代理工具Clash在MacOS系统的图形客户端，功能强大且支持多种代理协议。

获取ClashX

ClashX官网下载地址：https://github.com/yichengchen/clashX/releases 新手使用建议下载稳定版本，即版本号后标记为 Latest 的版本，下载列表中文件名为 ClashX.dmg 的文件即是macOS系统的安装包文件，文件列表中其余两个文件为软件的源代码压缩包，分别为 zip 格式和 tar.gz 格式。

添加 ClashX 配置订阅

添加订阅：

在Ghelper - 控制面板里点击左侧导航栏手机代理 ，找到ClashX订阅地址并点击复制。

打开ClashX软件，前往 用户中心 - 订阅 - Macos，复制 Clash 订阅链接。 点击状态栏中的 ClashX ，选择 配置>托管配置>管理。

单击弹窗左下角的 “添加” 。 4.粘贴您的订阅地址后单击"确认"按钮。

更新完毕订阅后，点击 ClashX 状态栏图标，将 “出站模式” 选为 “规则判断”，在选择自己喜欢的节点后，点击 “设置为系统代理” 即可开始使用

选择节点

轻点图标 >「GLOBAL」或 「Proxy」，根据您的需求选择可用节点。找到可用节点

轻点图标 >「GLOBAL」或 「Proxy」>「延迟测速 / Speed test」可找到可用节点。

不要选择「DIRECT」或「REJECT」节点，会导致断网。

选择代理模式

轻点图标 >「出站模式 / Proxy Mode」，根据您的需求选择代理模式：

全局连接 / Global：代理所有流量

适用于不依赖大陆服务的用户。 对国外流量效果非常好，大陆流量会被减速。

规则判断 / Rules：只代理国外流量

适用于同时使用国内外服务的用户。 大陆流量不会消耗套餐流量。 在大陆网站上查询 IP 得到的是本地 IP 地址。 在国外网站上查询 IP 得到的是代理 IP 地址。 分流规则无法做到全面且具有时效性，如果遇到以下情况，请尝试全局代理。

  • 无法打开国际网站；

  • 加载国际网站缓慢；

  • 无法解锁 Netflix / Spotify 等区域限制内容。

脚本模式 / Script：不可用

我们暂未提供脚本模式的配置文件，请不要选择此模式。

直接连接 / Direct：不代理任何流量

选择此模式将导致无法翻牆，与关闭 VPN 的效果一致。
